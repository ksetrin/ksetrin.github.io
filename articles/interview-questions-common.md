1. что значит error HTTP 409 - конфликт состояния
2. Что произойдёт, если выполнить тяжёлую операцию в UI-потоке? - ui зависнет(anr)
3. Какой компонент в React Native используется для оптимизированного отображения длинных списков? - flatlist (virtilizedlist)
4. Какой тип в TypeScript описывает функцию с аргументом string и возвратом number? - type Fn = (arg: string) => number
5. Какой объект Kubernetes используется для хранения конфиденциальных данных (пароли, ключи)? -  Secret
6. Как в TypeScript правильно типизировать пропсы компонента? -  Через interface или type и использование в React.FC<Props>
7. Что произойдёт, если в React вызвать setState несколько раз подряд с разными значениями? -  Обновления могут быть батчированы и выполнены одной операцией
8. Что такое Docker Image? - шаблон (снимок) для контейнера
9. Что вернёт useRef при первом вызове в компоненте? -  Объект с полем current, равным начальному значению
10. Какой стандарт безопасности чаще всего применяется в SOAP-сервисах? - WS-Security
11. Как в TypeScript обозначить необязательное свойство интерфейса? -  name?: string
12. Как правильно обновить состояние на основе предыдущего значения? -  setCount(prev => prev + 1)
13. Какой формат используется по умолчанию в gRPC для сериализации сообщений? - Protocol Buffers (Protobuf)
14. Какой хук React используется для управления побочными эффектами? - useEffect
15. Какое CSS свойство отвечает за выравнивание элементов по главной оси Flexbox? -  justify-content
16. Что произойдёт, если в React вернуть массив элементов без ключей? -  Может возникнуть неправильный рендер при изменении массива
17. Что произойдёт, если в useEffect указать пустой массив зависимостей? -  Эффект выполнится один раз после монтирования
18. Что делает React.memo в React Native? -  Мемоизирует компонент и предотвращает лишние рендеры
19. При деплое React Native backend-сервиса в Kubernetes нужно хранить токены для внешнего API. Где правильнее разместить их? -  В Kubernetes Secret
20. Что произойдёт, если передать новый объект в setState в функциональном компоненте? -  Состояние полностью заменится на новое значение
21. Чем OpenShift отличается от "чистого" Kubernetes? -  Имеет встроенные средства CI/CD и управление безопасностью
22. В Bitbucket создан Pull Request, но TeamCity билд упал. Какое действие правильнее? -  Исправить код и перезапустить билд, затем обновить PR
23. Какой тег HTML5 используется для семантической разметки основного содержимого страницы? - main
24. Какое преимущество имеет gRPC по сравнению с REST? -  Более компактная сериализация и поддержка двунаправленных стримов
25. Какой элемент описывает контракт SOAP-сервиса? - WSDL
26. Какой способ наиболее корректен для задания условного класса в React JSX? -  <div className={isActive ? "active" : "inactive"}></div>
27. Команда использует GitLab CI/CD для сборки образа React Native backend-а. Где правильнее хранить данные для подключения к приватному Docker Registry? -  В GitLab CI/CD Variables
28. Что делает CSS свойство flex: 1 в контейнере Flexbox? -  Заставляет элемент занимать всё доступное пространство пропорционально
29. Что произойдёт, если у FlatList не указать keyExtractor и объекты не имеют поля key? - будет использовать индекс как key (и warning)
30. Что произойдёт, если передать undefined в value контролируемого input в React? - станет неконтролируемым (warning)
31. Какой оператор в TypeScript используется для проверки наличия свойства в объекте? - in
32. Что делает метод useMemo в React Native? -  Мемоизирует вычисленное значение до изменения зависимостей
33. Что такое Pod в Kubernetes? -  Наименьшая единица деплоя, содержащая один или несколько контейнеров
34. Какая особенность distinguishes React Native от ReactJS? - ReactJS работает в DOM, React Native — с нативными UI-компонентами.
35. Что является основным отличием Virtual DOM от реального DOM? - Virtual DOM — это JS-объект в памяти, обновления дешевле, чем у реального DOM.
36. Как правильно типизировать состояние через useState в TypeScript? -  const [value, setValue] = useState([] as T)
37. Что делает хук useEffect без второго аргумента? - выполняется после каждого рендера
38. Какой хук в React Native используется для управления локальным состоянием функционального компонента? - useState
39. Какой HTTP метод чаще всего используется для обновления существующего ресурса в REST API? - PUT
40. Какой подход лучше использовать при интеграции React Native приложения с SOAP-сервисом? -  Использовать готовый SOAP-клиент с генерацией методов из WSDL
41. Какой тип используется в TypeScript для описания значения, которое может быть string или null? -  type T = string | null
42. Что делает метод useCallback в React/React Native? -  Возвращает мемоизированную версию функции для оптимизации рендера
43. В React Native приложении нужно отобразить данные в реальном времени (например, статус заказа). Какой протокол лучше выбрать? - WebSocket для real-time
44. Какой хук React используется для оптимизации вычислений при повторных рендерах? - useMemo
45. В React Native приложение нужно получать данные из REST API и отображать их в FlatList. Как правильно организовать логику? - Вызвать API в useEffect, сохранять данные в state и передать в FlatList
46. Какой файл используется для описания процесса сборки образа Docker? -  Dockerfile
47. Что означает идемпотентность в REST API? -  Повторный запрос приводит к одному и тому же результату
48. Какой формат сериализации данных обеспечит минимальный размер сообщений при передаче между сервисами? - Protocol Buffers (Protobuf)
49. Что возвращает useReducer в React? -  Кортеж [state, dispatch]
50. Какой тип импортируется из React для типизации children в TypeScript? -  ReactNode
51. Как выполнить запрос в фоновом потоке (когда приложение свернуто (background))? - Android: WorkManager / Headless JS и iOS: Background Fetch / Push Notification, а так же react-native-background-fetch, react-native-background-task
